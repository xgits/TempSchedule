GDC 2026 周三 Swag 行动计划

今天只做一件事：优先抢最容易断货、最容易爆火的 swag。周四周五再留给普通 booth、活动、party 和社群。

黄金时间：10:00–11:30 上午只抢限量 下午补漏 + 轻松扫货 优先级：Hyper3D → Xbox → NVIDIA → Dev Stream Labs → SideFX

你今天只要记住这 3 条

  1. 10:00–11:30 是黄金时间。 最容易断货的内容尽量压在这个时间段完成。
  2. 上午只做“抢限量”。 不要在一个 booth 聊太久，也不要为了普通贴纸排长队。
  3. 下午只做“补漏 + 轻松扫货”。 Commons、社群桌、普通 booth 和顺路活动都留到后面。

周三线性行动计划

按照这个节奏走，最省脑，也最不容易错过爆款。

09:00–09:45
进场准备

确保 badge、充电宝、背包空间、手机邮箱和相机都准备好。尽量提前到 South Hall / 入场动线附近，为 10 点开抢做准备。

  • 检查 badge、钱包、充电宝
  • 背包留出 swag 空间
  • 手机打开邮箱、相机、备忘录
  • 准备好一句话：Any swag today?
10:00–11:30
先抢最容易没的 5 个点

这段时间不要分心，先锁定最可能断货、最容易排爆的目标。

  1. Hyper3D：第一优先。先问规则，能登记/提交/排队就立刻做，先锁资格。
  2. Xbox / Microsoft：第二优先。重点盯 pin / 胸针类，通常最容易早段发完。
  3. NVIDIA：第三优先。核心是先做 giveaway / 抽奖登记，不要先花时间深聊。
  4. Dev Stream Labs：第四优先。有游戏化/抓娃娃/奖品机制，队伍可能突然变长。
  5. SideFX：第五优先。既然官方暗示要早点去，就尽量今天上午完成。
11:30–12:30
吃点东西 + 低压力补货

不要继续硬冲大热门。这个时段适合轻松扫一些稳定能拿的资料和小周边。

  • GDC Commons
  • IGDA / 社群组织桌
  • Roundtable / meetup 外围桌面资料
  • 小贴纸、小卡片、小徽章
12:30–14:30
扫中优先级 booth

走得轻松一点，优先看桌上有没有直接可拿的，没有再问一句是否有 swag。

  • Reddit
  • Meta
  • Red Hat
  • Sentry
  • Discord
  • Xsolla
  • Tencent（先问有没有 swag，再决定停留多久）
14:30–16:00
回刷上午错过的爆款点

这是最划算的二轮检查时间。不是为了重跑全部，而是捡回上午没成的关键目标。

  • 回看 Hyper3D
  • 回看 Xbox
  • 回看 NVIDIA
  • 回看 Dev Stream Labs
  • 回看 SideFX
16:00–18:00
轻松收尾

别再高强度冲了，改成查缺补漏、回刷喜欢的 booth、顺路看 Commons 或户外点。

  • 补拿漏掉的普通 swag
  • 社群区再扫一遍
  • 记下周四值得回刷的 booth
  • 给背包做一次整理
